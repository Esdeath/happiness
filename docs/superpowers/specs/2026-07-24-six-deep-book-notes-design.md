# 六本幸福主题书籍深度读书笔记设计

## 目标

为 2026-07-24 新增到书单的六本书分别撰写一篇深度读书笔记。每篇约 2500–3500 字，既准确交代全书主旨，也从本站关注的幸福、意义、关系、选择、工作与生活实践等角度展开分析。

六篇文章直接发布到 `src/content/posts/`，沿用现有读书笔记的排版、标签和相互链接方式。现有书单条目补充指向对应读书笔记的正文链接。

## 写作方法

采用“主题驱动的深度读书笔记”，不按章节平均复述，也不套用完全相同的小标题。每篇围绕一个清楚的核心问题组织材料，同时包含以下内容：

- 作者要回答的问题及全书核心论点。
- 相关理论的背景、证据类型和概念之间的关系。
- 四至六个值得保留的重点观点。
- 容易误解、过度推广或需要更新的部分。
- 能落实到具体生活场景的实践建议及适用边界。
- 适合的读者，以及不能用本书替代的专业支持。

转述使用自己的语言，不虚构原文引语、章节、页码或研究结论。事实性信息优先依据作者、出版社、原始论文或可信学术资料；无法核实的细节不写。

## 六篇主线

### 《象与骑象人》

核心问题是理性为什么常常无法直接命令情绪与直觉，以及一个“分裂的心智”如何获得较稳定的幸福。文章解释“象与骑象人”隐喻、道德与社会直觉、互惠和关系，并辨析“理性无用”这一常见误读。

### 《活出心花怒放的人生》

核心问题是积极心理学如何进入中国人的真实生活。文章围绕积极体验、优势、关系、意义与韧性，讨论工作压力、家庭关系、养育和社会文化情境，同时避免把结构性困境简化成个人心态问题。

### 《持续的幸福》

核心问题是塞利格曼为什么从“真实的幸福”转向“蓬勃”，以及 PERMA 比单一快乐指标多解释了什么。文章梳理五要素、优势与积极练习，也讨论测量、因果关系和把 PERMA 当作打分表的风险。

### 《思考，快与慢》

核心问题是直觉与审慎思考怎样共同决定选择，而判断偏差又如何影响生活质量。文章解释系统 1、系统 2、启发式、损失厌恶和前景理论，并区分教学模型与大脑中真实存在的两个系统，纳入相关研究的更新与争议。

### 《哈佛幸福课》

核心问题是人为什么经常预测错未来的幸福。文章围绕情感预测、影响偏差、聚焦错觉、心理免疫系统、记忆和未来想象展开，并说明这些偏差为什么既会误导选择，也帮助人从挫折中恢复。

### 《幸福的种子》

核心问题是心理学基础知识怎样帮助人认识自己并改善生活。文章用感知、记忆、情绪、社会关系、文化和积极心理学串起一张入门地图，强调心理学素养的价值，同时避免把心理学知识写成万能自助术。

## 验收标准

- 六篇文章各自处于 2500–3500 字的目标区间，并具有不同的论证主线。
- 标题、摘要、正文和对应书单推荐语彼此一致。
- 概念、作者信息和研究背景可核实，没有虚构引文与页码。
- 每篇都包含理论解释、必要辨析、实践建议和适用边界。
- 六个书单条目均链接到正确文章，六篇文章均链接回书单。
- `npm run build` 成功，内容 schema、图片、路由和 Pagefind 索引均可生成。

## 发布

只暂存六篇新文章和六个书单条目的链接修改。构建通过后使用一个聚焦的 `post:` 提交并推送当前分支，触发站点部署。
